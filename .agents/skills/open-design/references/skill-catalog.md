## Utility skills (run on demand)

| Skill | When to invoke |
|---|---|
| `critique` | After generation, when user says "review this" |
| `tweaks` | After generation, when user asks for small changes |
| `wireframe-sketch` | Before high-fi build, to validate structure |
| `design-brief` | When the user has only a vague idea |

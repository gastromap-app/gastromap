# 📋 План: Аудит Проекта и Следующие Шаги (V2)

Этот план фокусируется на переходе к целевой архитектуре (V2), интеграции с Supabase и реализации продвинутых функций админки. Основной упор сейчас: **Премиум UI/UX** и **Локализация**.

**Тип Проекта:** WEB (React + Vite + Supabase)

## 📞 Отложенная Задача: Голосовой Ассистент (Telnyx)
**Статус:** ⏸️ На паузе (Ждем регистрацию бизнес-аккаунта)
**Цель:** Внедрить `telnyx-toolkit` для голосового AI-общения (бронирование столиков, поддержка).

**Необходимые шаги для возвращения к задаче:**
1.  [ ] Зарегистрировать аккаунт Telnyx (нужна бизнес-почта/домен).
2.  [ ] Получить: `API Key`, `Public Key`, `Connection ID`.
3.  [ ] Купить номер телефона в Telnyx.
4.  [ ] Установить скилл `telnyx-toolkit` (с флагом `--force` или через проверку).
5.  [ ] Настроить входящий вебхук на ClawdTalk.

## 🛠 Технический Стек
- **Frontend:** React, Vite, Tailwind CSS, DaisyUI.
- **State:** Zustand (Auth, Избранное), TanStack Query (Локации).
- **Backend:** Supabase.
- **Автоматизация:** AG-Kit (Агенты), Google Maps Platform.
